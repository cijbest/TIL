### R 변수

- 스칼라 : 1차원 변수

  - 팩터 : 여러 개의 값 중에서 하나가 들어가도록 하는 것

    ```R
    a <- factor("A", c("A", "B", "C"))
    a
    is.factor(a)
    ```

- 벡터 : 배열

  - 한 가지 유형의 스칼라만 저장할 수 있음.

  - 벡터의 각 셀에는 이름을 부여할 수 있음

    ```R
    v1 <- c(1,2,3,4,5)
    sum(v1)
    ```

- 리스트 : k(키), v(값)
- 행렬 : 한 가지 유형의 스칼라만 저장할 수 있음.
- 배열 : 다차원 행렬
- 데이터 프레임 : 다중(데이터 타입) 행렬
